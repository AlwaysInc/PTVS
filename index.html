Python Tools for Visual Studio

Free, open source plugin that turns Visual Studio into a Python IDE

PTVS is a free, open source plugin that turns Visual Studio into a Python IDE.

It supports editing, browsing, IntelliSense, mixed Python/C++ debugging, remote Linux/MacOS debugging, profiling, IPython, and web development with Django and other frameworks.

From the Visual Studio 2017 or Visual Studio 2019 installer, select the Python or Data Science workload to add Python support to Visual Studio.

Need help? You can ask questions, file bugs or request features on our issue tracker on GitHub. Our documentation can be found here.

Designed, developed, and supported by Microsoft and the community.
